# KSHAP Brain measures (structural)

이 폴더에 있는 자료들은 KSHAP Cohort K 자료 중, brain imaging subcohort의 MRI data로부터 선택적으로 추출된, 뇌의 구조적 특징들을 정량화한 지표들입니다. 관련 문의는 kshap.yonsei@gmail.com 으로 보내주시기 바랍니다.

The data in this folder are quantitative indicators of structural brain features, selectively extracted from MRI data within the KSHAP Cohort K brain imaging subcohort. Please direct any related inquiries to kshap.yonsei@gmail.com.
