# KSHAP-Public
이 저장공간은 KSHAP (Korean Social Life, Health, and Aging Project) 데이터 중, 사회과학자료원(KOSSDA)을 통해 이미 공개된 설문 자료를 제외하고 다른 데이터의 사용을 희망하시는 분들을 대상으로 하는 안내 페이지입니다.

1) KSHAP 데이터를 사용한 socio-centric network 분석에 필요한 network ID 등 비공개 데이터 사용을 희망하시는 경우, Data Use Agreement Form 폴더의 국문 또는 영문 협약서를 작성하여 kshap.yonsei@gmail.com으로 제출해주시기 바랍니다. 단, 승인 여부는 연구진 내부 심의결과에 따라 결정되며, 협약서를 제출하시는 모든 분들께서 요청하시는 데이터가 전부 제공되지 않을 수 있습니다.
2) KSHAP에 대한 자세한 소개는 다음의 연구 설계 논문(study design paper)을 참고하시기 바랍니다: https://doi.org/10.1093/aje/kwad190
3) 기타 문의사항에 대해서는 kshap.yonsei@gmail.com 메일 계정으로 연락주시기 바랍니다.



This repository is an information page for those wishing to use data from the KSHAP (Korean Social Life, Health, and Aging Project) that has not yet been publicly released through Korea Social Science Data Archive(KOSSDA), excluding survey data already made available.

1) If you wish to use non-public data, such as network IDs required for socio-centric network analysis using KSHAP data, please complete the Korean or English version of the Data Use Agreement Form found in the Data Use Agreement Form folder and submit it to kshap.yonsei@gmail.com. However, approval is subject to internal review by the KSHAP research team, and not all requested data may be provided to every applicant.
2) For a detailed introduction to KSHAP, please refer to the following study design paper: https://doi.org/10.1093/aje/kwad190
3) For other inquiries, please contact us via the email account kshap.yonsei@gmail.com.
